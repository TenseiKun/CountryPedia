    About CountryPedia

    Edu

    What is this?

    CountryPedia is a web application built as a school project to practice frontend development, API integration, and UI design principles. It fetches live data from public APIs to show detailed information about every country in the world.

    How

    How it works

    The app uses the REST Countries API to fetch country data, the Time.now API for real-time world clocks, and Google Gemini as the AI chatbot engine. All data is fetched live using fetch() with async/await.

    Dev

    Tech used

    • Vanilla HTML, CSS, JavaScript
    • REST Countries API v3.1
    • Time.now World Time API
    • Google Gemini 2.5 Flash Lite
    • Google Fonts (Syne + DM Sans)
